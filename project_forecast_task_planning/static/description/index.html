Project Forecast Task Planning

Adds option to plan shifts by project's tasks.

Version: V16.1.0.2

Created and maintained by
MyOdoo.pl
assistance and support
kontakt@myodoo.pl

Description

Module's functionality:
* Adds option to assign shifts to tasks.
* Allows to group by tasks on shift views.
* Allows to see forecasted hours for each task.
* Adds task to shift notification.

Usage

Task field on shift's creation form.




Forecasted hours smart button on task form view.


Shift notification.


Installation

This module requires Odoos 'project_forecast', 'planning', 'project' modules to be installed.

Download → Unzip → Move to your addons directory → Update the apps list → Find "project_forecast_task_planning" module → Install module
